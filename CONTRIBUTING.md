## Confidentiality

This repository contains proprietary and confidential information belonging to Appshore LLP. Contributors must:

- Not share, publish, or disclose any code, documentation, or information from this repository to unauthorized parties.
- Not use any proprietary information gained through contributions for purposes outside of this project without written consent from Appshore LLP.
- Return or destroy all confidential materials upon request or upon termination of the contributor relationship.

## How to Contribute

1. Create a feature branch from `develop` (never from `main`).
2. Make your changes following the project's code conventions (see [CLAUDE.md](CLAUDE.md)).
3. Submit a pull request to `develop` with a clear description of the changes.
4. All contributions must pass CI checks before merging.
