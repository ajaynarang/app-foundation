SALLY Console is a dedicated administration and developer interface, separate from the main dispatch application, where fleet owners and administrators manage platform configuration, integrations, and developer tools. Think of it as the back office for your SALLY account — the place where you set up how the system behaves rather than where you use it day to day.

On the configuration side, Console is where you tune SALLY to match your operation. Set up alert thresholds and escalation timeouts, configure invoicing defaults like detention rates and payment terms, customize settlement parameters, and manage operations settings that control how the monitoring and route planning engines behave. Every plan has access to basic configuration through Console, so even Haul ($29/truck/month) customers can adjust their billing and fleet settings from day one.

On Fleet ($49/truck/month) and above, Console becomes the hub for managing integrations. Connect your Samsara account for ELD and telematics data, configure sync intervals, and monitor integration health — all from a single dashboard that shows connection status, last sync time, and any errors that need attention. On Freight Force (custom pricing), Console unlocks the full developer platform: generate and manage API keys with usage tracking, configure webhooks to push real-time events to your own systems, set up OAuth clients for third-party applications, and install MCP connectors that let AI assistants like Claude Desktop, ChatGPT, or Cursor interact with your SALLY data programmatically. Console also handles team and account management — user invitations, role assignments, plan details, and billing information — giving administrators a single place to manage everything that is not day-to-day dispatching.
