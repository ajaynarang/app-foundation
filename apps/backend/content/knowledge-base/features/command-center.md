The command center is the dispatcher's mission control — a real-time operations dashboard that puts everything happening across your fleet into a single view. Available on Fleet ($49/truck/month) and Freight Force plans, it brings together live driver positions, load statuses, active alerts, and fleet KPIs so dispatchers never have to switch between tabs or tools to understand what is going on.

At the top, a KPI strip shows the numbers that matter most: active routes in progress, on-time delivery percentage (color-coded green, yellow, or red so you can spot trouble at a glance), current HOS violation count, average fuel efficiency across the fleet, and driver availability for new assignments. Below that, active route cards display each route's real-time progress including the assigned driver and vehicle, current location, next stop, ETA status (on time, at risk, or late), and remaining HOS hours. These cards update continuously as monitoring data flows in, so the picture is always current rather than stale.

Priority-sorted alerts appear alongside the route cards with one-click acknowledge and resolve actions, letting dispatchers handle issues without leaving the dashboard. The system health panel shows integration status for connected services — Samsara ELD, QuickBooks, weather feeds — so you know immediately if a data source goes offline. Dispatchers can create shift notes to document important context, pin them for handoff to the next shift, and link them to specific routes or drivers. Quick action counts show loads waiting to be assigned and routes waiting to be planned, making it obvious where attention is needed. The command center is designed for the reality of dispatch work: fast decisions, multiple priorities, and zero tolerance for stale information.
