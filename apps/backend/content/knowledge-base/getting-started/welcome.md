# Welcome

This is a placeholder knowledge-base document for the platform starter.

Add your own product documentation as Markdown files under
`apps/backend/content/knowledge-base/`. The ingestion pipeline
(`pnpm seed:knowledge`) will chunk, embed, and index every document here so the
AI assistant can answer questions grounded in your content.

Each file supports frontmatter: `title`, `audience`, `category`.
