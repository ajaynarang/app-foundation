SALLY is designed to be a complete platform on its own — load management, billing, driver pay, route planning, and monitoring are all built in, so many fleets run everything from SALLY without any external tools. When you do need to connect to other systems, the integration options scale with your plan. The Fleet plan ($49/truck/month) includes Samsara ELD integration for real-time HOS data, GPS tracking, and driver status — this powers the route planning engine and continuous monitoring with live data instead of estimates. The Freight Force plan (custom pricing) opens up the full integration and developer platform: QuickBooks for bidirectional invoice and settlement sync, TMS connectors for companies migrating from or running alongside an existing TMS, custom integrations through the developer platform, API keys for programmatic access, webhooks for real-time event notifications, OAuth for secure third-party connections, and MCP tool access for building custom AI workflows. All integrations are configurable per tenant and run through SALLY's sync engine with full logging. The Haul plan ($29/truck/month) focuses on the core TMS experience without external integrations, which keeps things simple for smaller operations that do not yet need them.
