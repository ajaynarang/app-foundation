# Keyboard Shortcuts

SALLY supports keyboard shortcuts for common actions across the platform.

## Global Shortcuts

| Shortcut | Action |
|----------|--------|
| **Cmd+K** (Mac) / **Ctrl+K** (Windows) | Open Sally AI chat or search |
| **Escape** | Close the current sheet, dialog, or Sally chat |

## Form Shortcuts

| Shortcut | Action |
|----------|--------|
| **Cmd+Enter** (Mac) / **Ctrl+Enter** (Windows) | Submit the current form |
| **Tab** | Move to the next field |
| **Shift+Tab** | Move to the previous field |
| **Escape** | Close the form sheet (for view-only sheets; edit/create sheets require clicking X or Cancel) |

## Sheet Behavior

Sheets (the slide-out panels used for creating, editing, and viewing records) respond to keyboard input:

- **View-only sheets**: Close with Escape, clicking X, or clicking outside the sheet.
- **Edit and create sheets**: Close with Escape or clicking X/Cancel. Clicking outside the sheet does not close it (to prevent accidental data loss).

## Navigation

The sidebar is the primary navigation method. There are no global keyboard shortcuts for navigating between sections — use the sidebar links or Sally AI search (Cmd+K) to find what you need.

## Sally AI Chat

| Shortcut | Action |
|----------|--------|
| **Cmd+K** / **Ctrl+K** | Open Sally chat |
| **Enter** | Send message |
| **Escape** | Close Sally chat |

## Accessibility

SALLY supports standard browser accessibility features:

- Tab navigation through interactive elements
- Screen reader compatible labels on buttons and form fields
- High contrast support through the dark theme
- Minimum 44x44px touch targets on all interactive elements
