# Console Overview

SALLY Console is your platform administration hub — a separate application from the main Web App, focused on configuration, integrations, and developer tools.

## Accessing Console

Click the **Console** link in the Web App sidebar. Console opens in a new browser tab with seamless single sign-on — no separate login required. Admin and Owner roles have full Console access.

## What Console Is For

The Web App is where you dispatch loads and manage daily operations. Console is where you configure how SALLY works for your organization:

- **Configuration**: Set operational parameters (HOS rules, optimization defaults), alert thresholds and notification channels, and invoicing templates and branding.
- **Integrations**: Connect external systems like Samsara (ELD/telematics) and QuickBooks (accounting). Manage sync health and troubleshoot integration issues.
- **Developer Tools**: Create API keys, configure webhooks, register OAuth clients, and set up MCP connections for external AI assistants. Requires Freight Force plan.
- **Team Management**: Invite and manage team members, assign roles, handle invitations.
- **Plan & Billing**: View your current plan, feature entitlements, fleet and user limits, and manage billing.
- **Organization**: Manage company details, legal entity information, DOT/MC numbers, and contact information.

## Console Navigation

Console uses its own sidebar organized by section:

- **Configuration**: Operations, Alerts, Invoicing
- **Integrations**: Connections, Sync
- **Developer**: API Keys, Webhooks, OAuth Clients, AI Assistants
- **Team**: Members, Invitations
- **Account**: Plan & Billing, Organization

## When to Use Console vs. Web App

| Task | Where |
|------|-------|
| Create or manage loads | Web App |
| View fleet map and command center | Web App |
| Generate invoices and settlements | Web App |
| Connect Samsara or QuickBooks | Console |
| Configure alert thresholds | Console |
| Create API keys or webhooks | Console |
| Invite team members | Either (Console has more options) |
| View plan and billing | Console |
