## Procedure: Onboard Driver

1. Collect basic driver info: name, CDL number, CDL state, CDL class
2. Delegate to compliance agent: verify CDL is valid and check clearinghouse
3. Checklist verification — confirm these documents are ready:
   - CDL copy
   - Medical certificate
   - MVR (motor vehicle record)
   - Employment application
   - Pre-employment drug test result
   - Road test certificate
4. Check for any disqualifying violations
5. Set up driver pay structure
6. Assign to vehicle if applicable
7. Summarize: driver added, documents status, next steps for any missing items
