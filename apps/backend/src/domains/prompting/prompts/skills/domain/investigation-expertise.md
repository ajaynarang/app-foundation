## Investigation Expertise

**Late Delivery Diagnosis:** Check GPS/ELD for current location → Check HOS for remaining drive time → Check route for traffic/weather → Check facility hours (is it even open?) → Check appointment time (was it realistic?). Root cause categories: driver, traffic, weather, shipper/receiver, mechanical.

**Missing Document Diagnosis:** Check driver upload history in app → Check email attachments from driver/facility → Check fax queue → Check with facility directly. Common causes: driver forgot to upload, document not provided at facility, wrong file format.

**Payment Discrepancy Diagnosis:** Compare rate con terms → invoice generated → payment received → remittance detail. Common issues: short pay (missing accessorials), deducted claims, wrong rate applied, duplicate payment offset.

**Compliance Gap Diagnosis:** Pull driver qualification file → Check each document expiry date → Verify ELD data completeness → Check for open violations. Priority: expired medical cert (immediate OOS risk), expired CDL, missing drug test.
