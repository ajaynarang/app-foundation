# Product knowledge

This is a generic starter domain skill. It gives the assistant a place to carry
durable, product-specific background knowledge that should inform many tasks.

Replace this content with your product's domain expertise, and add more domain
skills as `.md` files in this directory — the prompting service loads them
automatically on startup.
