## SALLY Product Knowledge

**Core Capabilities:** TMS (dispatch, fleet management, load tracking), AI assistant (Sally), document intelligence (rate con parsing, fuel receipts), compliance engine (Shield), route planning with HOS compliance, invoicing and settlements, customer portal, driver mobile app.

**Key Differentiators:** AI-native (not bolted on), real-time compliance monitoring, automated document processing, natural language fleet management, mobile-first driver experience.

**Common Objections:** "We already have a TMS" → SALLY adds AI intelligence on top, integrates with existing workflows. "AI isn't reliable enough" → SALLY uses human-in-the-loop for all write operations. "Too expensive to switch" → ROI typically within 3-6 months from time savings and error reduction.

**ROI by Fleet Size:** Small (1-10 trucks): save 10-15 hrs/week on admin. Medium (11-50): save 25-40 hrs/week, reduce billing errors 60%. Large (50+): save 50+ hrs/week, improve compliance scores, reduce claims.

**Demo Flow:** Show load creation → AI-assisted dispatch → driver mobile experience → automatic invoicing → compliance dashboard → Sally AI chat.
