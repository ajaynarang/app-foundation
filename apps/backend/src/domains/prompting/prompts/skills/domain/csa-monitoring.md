## CSA Monitoring

**Score Breakdown:** Each BASIC scored as percentile (0-100%) compared to peer group. Lower is better. Scores based on 24 months of roadside inspections and crash data.

**Alert Thresholds:** Unsafe Driving: >65%. HOS Compliance: >65%. Vehicle Maintenance: >80%. Controlled Substances/Alcohol: >80%. Crash Indicator: >65%. Driver Fitness: >80%. Hazmat: >80%.

**What Triggers FMCSA Action:** High scores → warning letter → focused inspection → compliance review → new entrant audit (first 18 months).

**DataQ Challenges:** Dispute incorrect violations within 60 days. File at dataqs.fmcsa.dot.gov. Common challenges: wrong carrier cited, duplicate entries, incorrect severity weight. Success rate ~40% when well-documented.

**Inspection Weighting:** More recent inspections weighted more heavily. Severity matters: OOS (out-of-service) violations count more. Time weighting: 0-6 months = 3x, 6-12 months = 2x, 12-24 months = 1x.

**ISS Scores:** Inspection Selection System. 1-100 score. Higher = more likely selected for roadside inspection. Based on BASICs and registration data.
